# SpeakFlow - AI 语音输入工具

## 项目介绍

SpeakFlow 是一个基于 Web 技术的 AI 语音输入工具，通过浏览器实现语音到文字的实时转换功能。用户只需点击按钮开始录音，即可将语音实时转换为文字，并支持一键复制识别结果。

## 功能说明

- **录音控制**：提供"开始录音"和"停止录音"按钮，按钮状态根据录音状态动态变化（启用/禁用）
- **语音识别**：基于 Web Speech API 实现语音到文字的实时转换，支持中文（zh-CN）识别，实时显示识别结果
- **实时反馈**：识别过程中同时展示已确认的最终结果（final text）和尚未确认的临时结果（interim text）
- **文本复制**：一键将识别结果复制到剪贴板，并提供复制成功的视觉反馈
- **状态指示**：录音时提供脉冲动画状态指示灯，清晰展示当前录音状态
- **字数统计**：实时统计并显示已识别文字的字数
- **快捷键支持**：支持按 `Esc` 键快速停止录音
- **响应式设计**：适配桌面端与移动端不同屏幕尺寸
- **错误处理**：完善的错误处理机制，覆盖网络错误、麦克风权限、设备不存在等异常场景

## 安装与使用方法

### 环境要求

- 现代浏览器（推荐 Chrome 或 Edge，需支持 Web Speech API）
- 麦克风设备

### 使用方法

1. 克隆项目到本地：
   ```bash
   git clone <仓库地址>
   cd SpeakFlow
   ```

2. 直接使用浏览器打开 `index.html` 文件，或通过本地服务器运行：
   ```bash
   # 使用 Python 启动本地服务器
   python -m http.server 8080

   # 或使用 Node.js http-server
   npx http-server -p 8080
   ```

3. 浏览器打开 `http://localhost:8080`

4. 点击"开始录音"按钮，允许浏览器使用麦克风权限

5. 开始说话，识别结果将实时显示在文本区域

6. 点击"停止录音"结束识别，或按 `Esc` 键快速停止

7. 点击"复制文本"将识别结果复制到剪贴板

## 核心功能模块

### 1. 语音识别模块（script.js）

- 使用 `SpeechRecognition` 接口实现语音识别
- 配置 `continuous: true` 实现持续识别
- 配置 `interimResults: true` 实现实时中间结果显示
- 语言设置为 `zh-CN`（中文）
- 完善的错误处理，覆盖以下场景：
  - `no-speech`：未检测到语音
  - `audio-capture`：未找到麦克风设备
  - `not-allowed`：麦克风权限被拒绝
  - `network`：网络错误
  - `aborted`：识别被中止

### 2. 状态管理模块

- 录音状态（isRecording）控制按钮启用/禁用
- 状态指示灯动画（脉冲效果）
- 结果区域边框颜色动态变化

### 3. 剪贴板模块

- 优先使用 `navigator.clipboard.writeText()`（现代 API）
- 降级方案使用 `document.execCommand('copy')`（兼容旧浏览器）
- 复制成功后提供 Toast 提示和按钮状态变化反馈

### 4. UI/UX 设计

- 渐变色背景，现代简洁风格
- CSS 变量实现一致的设计系统
- 响应式布局，移动端按钮纵向排列
- 自定义滚动条样式
- 按钮 hover/active 状态过渡动画
