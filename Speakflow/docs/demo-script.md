# Demo 视频讲解脚本

## 1. 开场

大家好，这是我的作品 SpeakFlow 语音输入法。它的目标是帮助用户通过语音快速输入文本，适用于课堂笔记、会议纪要、聊天短句和写作草稿等场景。

## 2. 产品痛点

传统键盘输入在长文本记录时效率较低，而普通语音转文字工具往往只负责转写，缺少编辑命令、导出和效率反馈。SpeakFlow 希望在准确度、易用性、响应速度和成本之间取得平衡。

## 3. 功能展示

首先打开页面，可以看到浏览器支持检测、输入设置、文本编辑区和效率统计模块。

我选择中文普通话，然后点击开始语音输入。

示例语音：

“今天我们讨论语音输入法产品的核心功能句号换行第一点是提高文本输入效率句号第二点是降低使用成本句号”

可以看到文本被实时识别到输入区，并且自动处理标点和换行。

接下来展示语音命令：

“删除上一句”

可以看到系统删除了最后一句。然后说：

“复制全文”

系统会把文本复制到剪贴板。

最后点击下载 TXT，可以导出文本文件。

## 4. 技术说明

本项目使用 HTML、CSS 和 JavaScript 实现，语音识别基于浏览器 Web Speech API。这样可以避免后端服务器和付费云 API，MVP 阶段成本为 0，同时响应速度较快。

## 5. 创新点说明

项目不仅实现语音转文字，还加入了语音编辑命令、自动标点、输入效率统计和场景模式设计，使它更接近一个轻量语音输入法，而不是普通录音转写工具。

## 6. 结束

以上就是 SpeakFlow 的主要功能展示。后续可以扩展专业词库、接入更高精度模型、增加文本润色和离线识别能力。
