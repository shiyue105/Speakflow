## 三、代码质量检查

| 检查项 | 标准 | 验证方式 |
|--------|------|---------|
| 无全局变量 | 除 IIFE 外无全局污染 | 检查 window 对象 |
| 函数单一职责 | 每个函数职责清晰 | 代码审查 |
| 无 console.log | 生产代码无调试输出 | 全局搜索 |
| CSS 变量一致性 | 所有颜色使用 CSS 变量 | 检查 style.css |
| HTML 语义化 | 使用 section/article/footer 等标签 | 检查 index.html |
| 无障碍 | aria-live 状态区域 | 检查 index.html |
| 无外部请求 | Network 面板无 CDN/API 请求 | DevTools 检查 |
