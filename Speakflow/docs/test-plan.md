# 测试计划

## 功能测试

| 测试项 | 操作 | 预期结果 |
|---|---|---|
| 浏览器支持检测 | 打开页面 | Chrome/Edge 显示“可用” |
| 开始识别 | 点击开始语音输入 | 状态变为“正在聆听” |
| 停止识别 | 点击停止 | 状态变为“已停止” |
| 中文输入 | 说中文句子 | 文本进入输入区 |
| 英文输入 | 切换 English 并朗读英文 | 英文文本进入输入区 |
| 自动标点 | 说“句号/逗号/问号” | 转换为对应符号 |
| 换行命令 | 说“换行” | 插入换行 |
| 删除上一句 | 说“删除上一句” | 删除最后一句文本 |
| 复制全文 | 点击复制或说“复制全文” | 文本复制到剪贴板 |
| 下载 TXT | 点击下载 TXT | 生成 txt 文件 |
| 清空文本 | 点击清空或说“清空文本” | 输入区清空 |

## 异常测试

| 情况 | 预期处理 |
|---|---|
| 浏览器不支持 Web Speech API | 禁用开始按钮，显示不可用 |
| 用户拒绝麦克风权限 | 日志显示识别错误 |
| 无声音输入 | 页面保持运行，不崩溃 |
| 频繁开始/停止 | 主分支仍可运行，无重复启动崩溃 |

## 质量检查

- JS 代码按类封装，函数职责清晰。
- 页面无外部依赖，便于部署和复现。
- 主分支任意时刻可打开运行。
- README 明确列出依赖和原创部分。
